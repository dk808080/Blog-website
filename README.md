# Blog-website
A simple nodeJs application in which you can write blogs.

## Tech stack 
NodeJs<br/>
ExpressJs<br/>
EJs<br/>
MongoDB <br/>
Mongoose
## Available Scripts

In the project directory, you can run:
### `npm install`
to install all needed dependencies

### `node app.js`
to start the server

<br/>
Then navigate to your browser http://localhost:3000/ to view the app
